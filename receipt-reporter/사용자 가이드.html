출장 영수증 보고서 생성기

사용자 가이드 v1.2 (최종) | 설치 없이 바로 사용하세요

이 프로그램은?

출장에서 사용한 영수증 사진을 올리면, AI가 자동으로 읽어서 출장 보고서와 비용 지출 보고서를 만들어주는 프로그램입니다.

핵심 가치: 영수증 정리에 30분 이상 걸리던 작업을 5분 이내로 줄여줍니다.

지원 기능

  • 영수증 사진 일괄 업로드 (최대 50장)
  • AI OCR 자동 인식 (한국어, 영어, 일본어, 중국어 등 10개 언어)
  • 국내/해외 출장 모두 지원 — 복수 국가 환율 입력 가능
  • 비용 카테고리 자동 분류 (교통비, 숙박비, 식비, 접대비 등)
  • 세무 검증 자동 수행 (적격증빙, 부가세 공제, 가산세 위험)
  • 불필요한 영수증 제외 기능 — 잘못 올린 사진을 보고서에서 제외
  • 보고서 다운로드: Excel, Word, PDF 3가지 형식
  • 대시보드 — 이전 보고서 이력 관리 및 이어쓰기
  • 입력 내용 자동 임시 저장 (브라우저 종료해도 복원)
  • 인터넷 불필요 (최초 설치 후 완전 오프라인 동작)
  • 완전 무료 — API 비용, 구독료 없음

시작하기 (최초 1회)

1 폴더 확인 — receipt-reporter 폴더를 원하는 위치에 복사합니다.
2 실행 — 폴더 안의 run.bat을 더블클릭합니다.
3 초기 설정 (최초 1회만) — 필요한 라이브러리가 자동 설치됩니다. 몇 분 걸릴 수 있습니다.
검은 화면에 "라이브러리 설치 완료" 메시지가 나오면 성공입니다.
참고: 해외 출장 영수증을 처음 인식할 때 해당 언어의 OCR 모델이 자동 다운로드됩니다 (1회만, 수 분 소요).
4 브라우저 자동 열림 — 웹 브라우저에 프로그램 화면이 자동으로 열립니다.
안 열리면 브라우저 주소창에 http://127.0.0.1:8500 을 직접 입력하세요.
사전 준비: Python 3.11 이상이 PC에 설치되어 있어야 합니다.
Python이 없다면 IT팀에 설치를 요청하거나, python.org에서 다운로드하세요.
설치 시 "Add Python to PATH" 체크를 반드시 해주세요.

사용 방법 (4단계)

대시보드 (홈 화면)

프로그램을 열면 대시보드가 표시됩니다.
"+ 새 출장 보고서 작성" 버튼을 클릭하여 새 보고서를 시작하세요.
이전에 작성한 보고서가 있으면 목록에서 클릭하여 이어쓰기할 수 있습니다.

Step 1: 출장 정보 입력

출장자 이름, 출장지, 기간, 목적 등 기본 정보를 입력합니다.
* 표시된 항목만 필수입니다 (성명, 출장지, 기간, 목적). 나머지는 선택사항입니다.
입력한 내용은 자동으로 임시 저장되므로, 실수로 창을 닫아도 다시 열면 복원됩니다.
이전 입력이 복원된 경우 상단에 안내 바가 표시되며, "초기화" 버튼으로 새로 시작할 수 있습니다.
기본 결제수단 설정: 하단의 "기본 결제수단"에서 법인카드 또는 개인카드를 선택하세요.
이후 모든 영수증의 결제수단이 선택한 값으로 자동 설정됩니다. 개별 영수증에서 다른 수단으로 변경할 수 있습니다.
해외 출장인 경우:
1. "출장 유형"을 "해외 출장"으로 선택하세요.
2. 아래에 환율 입력란이 나타납니다.
3. 방문한 국가의 통화를 선택하고 환율을 입력하세요. (예: JPY → 9.2원)
4. 여러 국가를 방문했다면 "+ 통화 추가" 버튼으로 추가하세요.
5. 선택한 통화에 따라 영수증 인식 언어가 자동으로 설정됩니다.
   예) 엔화(JPY) + 위안화(CNY) 선택 → 일본어 + 중국어 + 영어 자동 인식

Step 2: 영수증 업로드

영수증 사진을 올리는 단계입니다. 3가지 방법이 있습니다:
  • 드래그 앤 드롭 — 파일을 업로드 영역에 끌어다 놓기
  • 클릭하여 선택 — 업로드 영역 클릭 후 파일 선택
  • Ctrl+A로 전체 선택 — 파일 선택 창에서 모든 영수증 한 번에 선택
촬영 팁: 영수증이 밝고 글씨가 선명하게 찍힐수록 인식 정확도가 높습니다.
접히거나 구겨진 영수증은 최대한 펴서 촬영하세요.
지원 형식: JPG, PNG, BMP, PDF | 파일당 최대 10MB | 최대 50장

Step 3: 인식 결과 검토 (저신뢰도 우선)

AI가 영수증을 읽은 결과를 확인합니다. 신뢰도가 낮은 영수증부터 먼저 표시됩니다.
신뢰도가 높은 영수증은 이 단계에서 별도 확인 없이 넘어가도 됩니다.
왼쪽에 원본 영수증, 오른쪽에 인식된 데이터가 표시됩니다.

신뢰도 색상:
  • ● 녹색 (높음) — 정확하게 인식됨 (확인 불필요)
  • ● 노란색 (보통) — 대부분 맞지만 확인 권장
  • ● 빨간색 (낮음) — 인식 실패 가능성 높음, 반드시 확인
  • ● 회색 (제외됨) — 보고서에서 제외된 영수증
잘못 인식된 항목은 직접 클릭하여 수정할 수 있습니다.
통화와 결제수단은 Step 1에서 설정한 값이 기본 적용되어 있습니다.
영수증 제외 기능: 잘못 올리거나 불필요한 영수증은 각 영수증 하단의 "이 영수증 제외하기" 버튼을 클릭하면 보고서에 포함되지 않습니다.
제외한 영수증은 언제든 "포함하기"로 다시 복원할 수 있습니다.

Step 4: 보고서 생성 및 다운로드

최종 요약(총 비용, 카테고리별 지출, 적격증빙 수취율 등)을 확인하고,
원하는 형식으로 보고서를 다운로드합니다.

"전체 영수증 목록"을 펼치면 모든 영수증을 한눈에 확인할 수 있습니다.
여기서 카테고리와 결제수단을 드롭다운으로 직접 변경하거나, 불필요한 영수증을 제외할 수 있습니다.
Step 3에서 검토하지 않은 신뢰도 높은 영수증도 이 목록에서 최종 확인하세요.
  • Excel (.xlsx) — 데이터 편집이 필요할 때 (가장 추천)
  • Word (.docx) — 회사 양식에 맞춰 수정 후 제출할 때
  • PDF (.pdf) — 인쇄하거나 수정 없이 제출할 때
다운로드된 파일은 data/output/ 폴더에도 자동 저장됩니다.

다른 PC에서 사용하기

1 receipt-reporter 폴더 전체를 복사합니다 (USB, 공유 폴더 등)
2 새 PC에 붙여넣기 합니다
3 run.bat을 더블클릭합니다
4 최초 1회 라이브러리 자동 설치 후 사용 가능
새 PC 요구사항:
  • Windows 10 이상
  • Python 3.11 이상 설치 (PATH에 추가되어 있어야 함)
  • 인터넷: 최초 실행 시에만 필요 (라이브러리 + OCR 모델 다운로드). 이후 오프라인 사용 가능

자주 묻는 질문 (FAQ)

Q. run.bat을 더블클릭하면 창이 바로 닫혀요
Python이 설치되어 있는지 확인하세요. 명령 프롬프트에서 python --version을 입력하여 버전이 나오면 정상입니다.
Q. 브라우저가 안 열려요
검은 창에 "http://127.0.0.1:8500"이 보이면, 이 주소를 브라우저에 직접 입력하세요.
Q. 영수증 인식이 잘 안 돼요
  • 사진이 충분히 밝고 선명한지 확인
  • 해외 출장인 경우 Step 1에서 올바른 통화(환율)를 선택했는지 확인
  • 처음 해외 영수증 인식 시 모델 다운로드에 수 분 소요될 수 있음
  • 인식 실패 시 Step 3에서 직접 수동 입력 가능
Q. 잘못 올린 영수증을 빼고 싶어요
Step 3에서 해당 영수증의 "이 영수증 제외하기" 버튼을 클릭하세요. 제외된 영수증은 보고서에 포함되지 않으며, 언제든 다시 포함시킬 수 있습니다.
Q. 비용이 발생하나요?
완전 무료입니다. 모든 처리는 PC 내에서 이루어지며, 외부 서버나 유료 API를 사용하지 않습니다.
Q. 이전에 만든 보고서를 다시 보고 싶어요
대시보드(홈 화면)에서 이전 보고서 목록을 확인할 수 있습니다. data/output/ 폴더에도 파일이 저장되어 있습니다.
Q. 프로그램을 종료하려면?
검은 창(명령 프롬프트)을 닫으면 프로그램이 종료됩니다.
Q. 여러 나라를 출장 갔는데 영수증 언어가 다양해요
Step 1에서 방문한 국가의 통화를 모두 추가하세요. 통화에 따라 인식 언어가 자동으로 설정됩니다. 예: 엔화 + 달러 선택 → 일본어 + 영어 자동 인식

문의사항은 관리자에게 연락하세요
